# Data Flow Diagram (DFD) Level 1 - ระบบร้านค้าออนไลน์ (My Shop)

## 👤 เอนทิตีภายนอก (External Entities)

คือผู้ที่เกี่ยวข้องกับระบบ ประกอบด้วย:

- **Admin** (ผู้ดูแลระบบ)
- **Customer** (ลูกค้า/สมาชิก)

## 🔵 กระบวนการทำงาน (Processes)

**1.0 จัดการการเข้าสู่ระบบและข้อมูลสมาชิก** (Authentication & User Management)
**2.0 จัดการข้อมูลสินค้าและหมวดหมู่** (Product Management)
**3.0 จัดการตะกร้าสินค้า** (Cart Management)
**4.0 จัดการคำสั่งซื้อ** (Order Management)
**5.0 จัดการการชำระเงิน** (Payment Management)
**6.0 จัดการแบนเนอร์และการโปรโมท** (Banner Management)

## 🗄️ แหล่งเก็บข้อมูล (Data Stores)

**D1** ข้อมูลสมาชิก (Users)
**D2** ข้อมูลสินค้าและสต็อก (Products)
**D3** ข้อมูลหมวดหมู่สินค้า (Product Types)
**D4** ข้อมูลคำสั่งซื้อและรายละเอียด (Orders & Order Details)
**D5** ข้อมูลการชำระเงิน (Payments)
**D6** ข้อมูลแบนเนอร์ (Banners)
**D7** ข้อมูลตะกร้าสินค้า (Cart)

## ➡️ ข้อมูลที่ไหลในระบบ (Data Flows)

### Process 1.0 - จัดการการเข้าสู่ระบบและข้อมูลสมาชิก

- Customer → Process 1.0: Username/Password
- Admin → Process 1.0: Username/Password
- Customer → Process 1.0: ข้อมูลการสมัครสมาชิก
- Customer → Process 1.0: ข้อมูลการลืมรหัสผ่าน
- Process 1.0 → Customer: ผลการยืนยันตัวตน / สถานะ Login
- Process 1.0 → Customer: ข้อมูลโปรไฟล์ลูกค้า
- Process 1.0 ↔ D1: อ่าน/บันทึกข้อมูลสมาชิก
- Admin → Process 1.0: ข้อมูลการจัดการสมาชิก (ดู/แก้ไข)

### Process 2.0 - จัดการข้อมูลสินค้าและหมวดหมู่

- Admin → Process 2.0: ข้อมูลสินค้า และสต็อก
- Admin → Process 2.0: ข้อมูลหมวดหมู่สินค้า
- Process 2.0 → Customer: ข้อมูลสินค้า (แสดงรายการ/รายละเอียด)
- Process 2.0 ↔ D2: อ่าน/บันทึกข้อมูลสินค้าและสต็อก
- Process 2.0 ↔ D3: อ่าน/บันทึกข้อมูลหมวดหมู่สินค้า

### Process 3.0 - จัดการตะกร้าสินค้า

- Customer → Process 3.0: ข้อมูลสินค้าในตะกร้า (เพิ่ม/ลด/ลบ)
- Process 3.0 → Customer: รายการสินค้าในตะกร้า
- Process 3.0 ↔ D7: อ่าน/บันทึกข้อมูลตะกร้าสินค้า
- Process 3.0 → D2: อ่านข้อมูลสินค้า (ราคา/รูปภาพ)

### Process 4.0 - จัดการคำสั่งซื้อ

- Customer → Process 4.0: ข้อมูลยืนยันการสั่งซื้อ (ที่อยู่/ชื่อ/เบอร์โทร)
- Process 4.0 → Customer: ใบเสร็จ / ประวัติการสั่งซื้อ
- Process 4.0 → Admin: รายการคำสั่งซื้อที่ส่งไปยังผู้ดูแลระบบ
- Admin → Process 4.0: หมายเลขพัสดุ / สถานะการจัดส่ง
- Process 4.0 ↔ D4: อ่าน/บันทึกข้อมูลคำสั่งซื้อและรายละเอียด
- Process 4.0 → D2: อัปเดตสต็อกสินค้า (หักจำนวนเมื่อสั่งซื้อสำเร็จ)
- Process 4.0 → D7: ลบข้อมูลตะกร้าหลังสั่งซื้อสำเร็จ

### Process 5.0 - จัดการการชำระเงิน

- Customer → Process 5.0: ข้อมูลการชำระเงิน และหลักฐานการโอนเงิน (สลิป)
- Process 5.0 → Customer: สถานะการชำระเงิน
- Admin → Process 5.0: ยืนยัน/ปฏิเสธการชำระเงิน
- Process 5.0 ↔ D5: อ่าน/บันทึกข้อมูลการชำระเงิน

### Process 6.0 - จัดการแบนเนอร์และการโปรโมท

- Admin → Process 6.0: ข้อมูลแบนเนอร์โฆษณา (เพิ่ม/แก้ไข/เปิด-ปิด)
- Process 6.0 → Customer: แบนเนอร์โฆษณา (แสดงหน้าแรก)
- Process 6.0 ↔ D6: อ่าน/บันทึกข้อมูลแบนเนอร์
