# Data Flow Diagram (DFD) Level 1 - ระบบร้านค้าออนไลน์ (My Shop)

## 👤 เอนทิตีภายนอก (External Entities)

คือผู้ที่เกี่ยวข้องกับระบบ ประกอบด้วย:

- **Admin** (ผู้ดูแลระบบ)
- **Customer** (ลูกค้า/สมาชิก)

## 🔵 กระบวนการทำงาน (Processes)

**1.0 จัดการการเข้าสู่ระบบและข้อมูลสมาชิก** (Authentication & User Management)
**2.0 จัดการข้อมูลสินค้าและหมวดหมู่** (Product Management)
**3.0 จัดการตะกร้าสินค้า** (Cart Management)
**4.0 จัดการคำสั่งซื้อ** (Order Management)
**5.0 จัดการการชำระเงิน** (Payment Management)
**6.0 จัดการแบนเนอร์และการโปรโมท** (Banner Management)

## 🗄️ แหล่งเก็บข้อมูล (Data Stores)

**D1** ข้อมูลสมาชิก (Users)
**D2** ข้อมูลสินค้าและสต็อก (Products)
**D3** ข้อมูลหมวดหมู่สินค้า (Product Types)
**D4** ข้อมูลคำสั่งซื้อและรายละเอียด (Orders & Order Details)
**D5** ข้อมูลการชำระเงิน (Payments)
**D6** ข้อมูลแบนเนอร์ (Banners)

## ➡️ ข้อมูลที่ไหลในระบบ (Data Flows)

นี่คือข้อความที่อยู่บนเส้นลูกศรทั้งหมดที่เชื่อมต่อการทำงานต่างๆ:

- Username/Password
- ข้อมูลการสมัครสมาชิก
- ข้อมูลโปรไฟล์ลูกค้า
- ข้อมูลสินค้า และสต็อก
- ข้อมูลหมวดหมู่สินค้า
- ข้อมูลสินค้าในตะกร้า
- ข้อมูลยืนยันการสั่งซื้อ
- รายการคำสั่งซื้อที่ส่งไปยังผู้ดูแลระบบ
- ข้อมูลการชำระเงิน และหลักฐานการโอนเงิน (สลิป)
- สถานะการชำระเงิน
- หมายเลขพัสดุ / สถานะการจัดส่ง
- ใบเสร็จ / ประวัติการสั่งซื้อ
- ข้อมูลแบนเนอร์โฆษณา
