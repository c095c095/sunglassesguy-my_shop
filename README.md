# My Shop - ระบบร้านค้าออนไลน์สำหรับอุปกรณ์ตกปลา

My Shop เป็นแพลตฟอร์มเว็บไซต์อีคอมเมิร์ซ (E-commerce) ที่พัฒนาขึ้นมาสำหรับการขายอุปกรณ์ตกปลาโดยเฉพาะ ตัวระบบถูกออกแบบโดยใช้ PHP แบบ Native ร่วมกับระบบ Routing ที่เขียนขึ้นเอง ฐานข้อมูล MySQL และใช้ Bootstrap 5 สำหรับการตกแต่งหน้าบ้าน (Frontend) พร้อมฟีเจอร์อื่นๆ เช่น Swiper สำหรับการแสดงผลแบบ Carousel

## เทคโนโลยีที่ใช้ (Tech Stack)

* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript (Vanilla), Swiper.js
* **Backend:** PHP 8+ (Custom Routing, Database queries, Business logic)
* **Database:** MySQL / MariaDB
* **Security:** ระบบจัดการสิทธิ์การเข้าถึง (Role-based access control: Admin, Customer, Guest)

## โครงสร้างโปรเจกต์ (Project Structure)

```text
my_shop/
├── admin/              # ระบบจัดการหลังบ้านสำหรับผู้ดูแลระบบ (Admin Panel)
├── assets/             # ไฟล์ CSS, JavaScript, รูปภาพ และไลบรารีต่างๆ (Bootstrap, Swiper)
├── core/               # การตั้งค่า (Configuration), ฟังก์ชันช่วยเหลือ (Helpers) และ Services 
├── database/           # ไฟล์โครงสร้างและข้อมูลของฐานข้อมูล (.sql)
├── instruction/        # เอกสารและคู่มือการใช้งาน/โครงสร้างของระบบ
├── layouts/            # ส่วนประกอบของหน้าเว็บ (Navbar, Footer)
├── pages/              # หน้าเว็บไซต์ส่วนของลูกค้า (Home, Products, Cart, Profile ฯลฯ)
└── upload/             # โฟลเดอร์สำหรับเก็บรูปภาพที่ถูกอัปโหลด (Banner, Payment, Product)
```

## ฟีเจอร์หลัก (Key Features)

### สำหรับลูกค้า (Customer)
* **ดูสินค้าและค้นหา:** หน้าแรก (Home) มีแบนเนอร์โปรโมชันและหมวดหมู่สินค้า สามารถกดดูรายละเอียดและตรวจสอบสถานะสินค้าคงคลังได้
* **ระบบสมาชิก:** สามารถสมัครสมาชิก เข้าสู่ระบบ แก้ไขข้อมูลส่วนตัว และดูประวัติการสั่งซื้อได้
* **ระบบตะกร้าสินค้า (Cart):** เลือกเพิ่ม/ลดจำนวนสินค้า หรือลบสินค้าออกจากตะกร้าได้ โดยระบบจะตรวจสอบจำนวนสินค้าคงคลัง (Stock) อัตโนมัติ
* **การชำระเงินและการจัดส่ง:** เมื่อยืนยันคำสั่งซื้อ จะคำนวณค่าจัดส่งและยอดรวมให้ ลูกค้าสามารถอัปโหลดสลิปโอนเงิน (Bank Transfer) ผ่านระบบได้

### สำหรับผู้ดูแลระบบ (Admin)
*(จำกัดสิทธิ์เฉพาะผู้ใช้ระดับ Admin เท่านั้น)*
* **Dashboard:** ดูภาพรวมของยอดขาย จำนวนคำสั่งซื้อ สินค้าที่หมดสต็อก และรายการรอตรวจสอบการชำระเงิน
* **จัดการคำสั่งซื้อ (Order Management):** ตรวจสอบสลิปการโอนเงิน และอัปเดตสถานะคำสั่งซื้อ (เช่น รอชำระเงิน, จ่ายแล้ว, จัดส่งแล้ว, ยกเลิก)
* **จัดการสินค้าและสต็อก (Catalog):** เพิ่ม/แก้ไข/ลบ ข้อมูลสินค้า หมวดหมู่สินค้า และอัปเดตจำนวนสินค้าคงคลังได้อย่างละเอียด
* **จัดการเนื้อหาและผู้ใช้งาน (Users & Content):** จัดการสิทธิ์การใช้งานของลูกค้า และจัดการรูปภาพแบนเนอร์บนหน้าแรกของเว็บ

## วิธีการติดตั้งและใช้งาน (Getting Started)

1. **เตรียมสภาพแวดล้อม (Environment):** 
   ให้แน่ใจว่าคุณติดตั้งโปรแกรมจำลองเซิร์ฟเวอร์ เช่น XAMPP, MAMP หรือ WAMP และใช้ PHP เวอร์ชัน 8 ขึ้นไป
2. **คัดลอกไฟล์ลงเซิร์ฟเวอร์:** 
   นำโฟลเดอร์ `my_shop` ไปวางไว้ในโฟลเดอร์ `htdocs` (สำหรับ XAMPP) หรือ `www` (สำหรับ MAMP/WAMP)
3. **ตั้งค่าฐานข้อมูล:** 
   * เปิด phpMyAdmin หรือเครื่องมือจัดการฐานข้อมูลของคุณ
   * สร้างฐานข้อมูลใหม่ (เช่น `myshop_db`)
   * Import ไฟล์ ฐานข้อมูลจากโฟลเดอร์ `database/` (เช่น `myshop_v1.2.0.sql`) เข้าไปในฐานข้อมูลที่สร้างขึ้น
4. **ตั้งค่าการเชื่อมต่อฐานข้อมูล:** 
   เข้าไปแก้ไขข้อมูลการเชื่อมต่อฐานข้อมูล (Database Credentials) ในไฟล์ `core/config/config_database.php`
5. **เข้าสู่ฟังก์ชันการใช้งาน:** 
   * **หน้าร้าน:** ไปที่ `http://localhost/my_shop/`
   * **หน้าแอดมิน:** ไปที่ `http://localhost/my_shop/admin/`
